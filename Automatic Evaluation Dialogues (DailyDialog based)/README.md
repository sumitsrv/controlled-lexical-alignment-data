# Automatic Evaluation Dialogues

**10,545 dialogues** generated using DailyDialog subset with varying alignment weights.

## Data Source

Modified versions of [DailyDialog](https://aclanthology.org/I17-1099/) dialogues:
- **Turns 1–3**: Retained from original DailyDialog
- **Turn 4+ (Speaker 2)**: Generated by LLM with controlled alignment weights
- **All Speaker 1 turns**: Retained from original DailyDialog

## Models

| Model | Dialogues | Weight Range |
|-------|-----------|--------------|
| BlenderBot-3B | 1,425 | 25–1000 |
| DialoGPT-small | 1,330 | 25–1000 |
| Llama-2-7b-chat | 3,895 | 25–7500 |
| Phi-3.5-mini-instruct | 3,895 | 25–7500 |

## Structure

```
├── alignment_data_20250329.tsv    # Aggregated metrics for all dialogues
└── models/{Model}/{Weight}/
    └── {dialogue_id}/
        ├── dialogue.txt           # The dialogue
        └── output/                # Computed alignment metrics
```

## File Formats

### dialogue.txt
```
Human 1:  [utterance]    # From DailyDialog
Human 2:  [response]     # Turn 1-3: DailyDialog, Turn 4+: Generated
```

### alignment_data_20250329.tsv
Key columns: `EV` (Expression Variety), `ER` (Expression Repetition), `Voc. Overlap`, `ENTR` (Entropy), `mname` (model), `weight`, `dialogue_count`

### output/ files
| File | Description |
|------|-------------|
| `metrics-speaker-independent.tsv` | Dialogue-level metrics |
| `metrics-speaker-dependent.tsv` | Per-speaker metrics |
| `dialogue_txt-lexicon.tsv` | Shared lexicon |

## Generation Parameters
- **History length**: 6 turns
- **Top-k sampling**: k=20
